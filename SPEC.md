# 银龄相馆 - AI 形象照应用规格文档

## 1. 项目概述

- **项目名称**: 银龄相馆 (Silver Portrait Studio)
- **项目类型**: Web 应用 (Next.js)
- **核心功能**: 为老年人提供 AI 形象照生成服务（证件照、节日祝福照、遗像）
- **目标用户**: 50-75岁老年人及其子女

## 2. 技术架构

### 技术栈
- **前端**: Next.js 14 + Tailwind CSS
- **后端**: Next.js API Routes
- **AI**: Gemini 3 Pro Image (via aihubmix)
- **存储**: Cloudflare R2
- **部署**: Dokploy

### 基础设施配置
- **域名**: ai-portrait.aipixbox.com
- **R2 Bucket**: st1
- **R2 公共域名**: https://st1.aipixbox.com

## 3. 功能规格

### 3.1 核心功能

#### 首页
- 大字体欢迎语
- 语言切换 (中文/English)
- 照片类型选择入口
- 操作流程说明

#### 照片类型
1. **证件照** - 蓝底/白底/红底
2. **节日祝福照** - 春节/中秋/生日
3. **遗像** - 遗照专用（高清修复）

#### 上传流程
1. 选择照片类型
2. 上传原图（支持拍照或相册）
3. AI 处理生成
4. 预览结果
5. 下载保存

### 3.2 UI/UX 要求
- **字体**: 至少 18px 基础字号
- **按钮**: 大尺寸 (高度 ≥ 56px)
- **对比度**: 高对比度配色
- **语言**: 中英文双语支持

### 3.3 API 端点

```
POST /api/generate - AI 生成形象照
GET  /api/status/:id - 查询生成状态
```
